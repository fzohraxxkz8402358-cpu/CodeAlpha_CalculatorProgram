# 🧮 C Basic Calculator

A simple command-line calculator developed in **C** that performs basic arithmetic operations. This project demonstrates the use of functions, conditional statements, user input, and basic programming concepts in C.

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* Input validation for division by zero
* Simple and user-friendly command-line interface

## 🛠️ Technologies Used

* C Programming
* GCC Compiler (MinGW)
* Visual Studio Code

## 🚀 Getting Started

### Prerequisites

* GCC Compiler
* Visual Studio Code or any C IDE

### Compile

```bash
gcc calculator.c -o calculator
```

### Run

```bash
./calculator
```

On Windows:

```bash
calculator.exe
```
